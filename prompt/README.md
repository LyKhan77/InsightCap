# InsightCap Prompt Flow

These prompts are designed to guide the project phase by phase using the flow:

`Start Phase -> Debugging -> Succeed -> Next Phase`

All prompt content is written in English because the project language is English.

## Files

- `phase-1-core-engine.md`
- `phase-2-api-layer.md`
- `phase-3-web-app.md`

## Suggested Usage

1. Start with the `Start Phase` prompt for the current phase.
2. If implementation issues appear, use the `Debugging` prompt for that phase.
3. Once the phase is stable, use the `Succeed` prompt to finalize and validate the outcome.
4. Then use the `Next Phase` prompt to transition into the next phase cleanly.

## Notes

- Development environment assumes Apple Silicon M4.
- Production target assumes Linux + NVIDIA GPU.
- Project reference: `https://x.com/HuggingModels/status/2036874146077131225`
